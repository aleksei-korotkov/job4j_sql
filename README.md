# job4j_sql

Учебный проект по изучению SQL и PostgreSQL.

## Цель
создания и изменения структуры таблиц;
настройки ограничений;
работы с внешними ключами и связями между таблицами.

## Содержимое
Скрипты для создания схемы БД;
Примеры использования команд;
Проектирования связей «один-ко-многим» и «многие-ко-многим».

## Технологии
- **СУБД**: PostgreSQL
- **Инструменты**: psql, DBeaver, IntelliJ IDEA
